This project entails web design, editing, and aesthetics using the Dreamweaver software (free version). Under this project, the previous mini projects were modified in terms of layout, coloring, insertions etc. Given that I was a beginer when it comes to Dreamweaver applications, I had to go through some basic tutorials that are provided in the course notes as well as those provided free on the internet. After abreasting myself with the operations of the software, I downloaded it. I then copied the original versions of the previous mini projects and saved them in a different folder. I opened those saved projects on the dreameaver and subsequently polished them using the various tools. The table below contains the original versions and the polished versions of the projects.

Though prvious exercises that entailed painstaking and time consuming writing of synthax in the notepad provided basic understanding of web designs, the dreamweaver exercise advances the skills and knowledge in a simple manner.
